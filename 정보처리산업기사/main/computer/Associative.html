연관기억장치

  • 연관기억장치는 기억장치에서 자료를 찾을 때 주소에 의해 접근하지 않고, 기억된 내용의 일부를 이용하여 Access 할 수 있는 기억장치로, CAM(Content Addressable Memory )라고도 한다
  • 보조기억장치 분류 DASD, SASD(공사중)

기억장치의 특성을 결정하는 요소

  • 기억용량
  • Access Time
    1. 기억장치에서 읽기 요청이 발생하고 요구한 정보를 꺼내서 사용 가능할 때 까지의 시간
    2. 사용 가능할때 : 메모리 버퍼 레지스터에 자료가 옮겨졌을 떄
    3. Access Time = Seek Time(트랙을 찾는시간) + Latency Time[Search Time](섹터를 찾는데 걸 리는 시간, 회전시간) + Transmission Time(트랙과 섹터를 찾은 후 전송하는데 걸리는 시간)
  • Cycle Time : 기억장치에서 읽기 신호를 보낸후 다시 신호를 보낼 수 있을 때 까지의 시간
  • Bandwidth(대역폭, 전송률)
    1. 메모리로부터 또는 메모리로 향해서 1초동안 전송되는 최대한의 정보량 기억장치의 자료 처리 속도를 나타내는 단위
    2. 메모리 워드의 길이가 작을수로 대역폭이 좋다
    3. 전송 단위 : Baud(보) = bps(초당 전송가능 비트수)

기억장치의 구분

  • 1. 내용의 보존 여부
    1. 파괴성 메모리(Destructive Memory) : 판독 후 저장된 내용이 파괴되는 메모리 , 파괴된 내용을 재생시키기 위한 재저장 시간(Restoration Time)이 필요하다.
      자기 코어
    2. 비파괴성 메모리 : 판독 후에도 저장된 내용이 그대로 유지
      자기 코어를 제외한 모든 기억장치
  • 2. 전원 단절 시 내용 소멸 여부
    1. 휘발성 메모리(Volatile Memory) : 전원 차단시 모든 정보가 지워짐
      RAM
    2. 비휘발성 메모리 : 전원 차단시에도 기억된 정보가 보존
      ROM, 자기코어, 보조기억장치
  • 3. 재충전 여부
    1. 공사중
  • 4. 접근 방식
    1. 공사중
