자료 구조

공사중

자료구조의 분류

  • 선형 구조 : 데이터를 순차적으로 나열시킨 형태
    • 리스트(선형, 연결), 스택Stack, 큐Queue, 데큐Deque
  • 비선형 구조 : 트리, 그래프
  • 큐 : 모든 삽입(insert)은 한쪽 끝에서 발생하고 모든 삭제는 다른 한쪽에서 발생하는 순서 리스트(ordered list)

관련 용어

  • 정렬(sort) : 자료를 일정한 순서에 의해 나열하는것
  • 검색(search) : 기억장치 내의 자료를 찾는 것
  • 파일 편성 : 자료를 기억 매체에 저장할 때의 파일 구조
  • 인덱스 : 파일에서 특정 자료를 빠르게 찾기 위한 색인표
